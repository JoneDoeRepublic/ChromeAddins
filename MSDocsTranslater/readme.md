# Microsoft Docs Translater
## Summary
Microsoft公式のDocsの日本語版ページと英語版ページの遷移を補助するChromeの拡張機能です。

## How to use
1. 本アドインのクローンをダウンロードしてください。
2. srcフォルダをChromeの拡張機能の管理から追加してください。
3. DocsページでShiftを押すと、言語が切り替わります。
